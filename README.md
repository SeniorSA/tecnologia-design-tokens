Design tokens created for BPM Suite products. 

A library using:
- `style-dictionary`
- `gulp` to post CSS processing
- `SASS`

Distributed via:
- [NPM](https://www.npmjs.com/package/@seniorsistemas/tecnologia-design-tokens)
- CDN
